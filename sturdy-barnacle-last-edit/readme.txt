**Welcome**
Thanks for installing Simply Show Last Edit Date!

**How to use this plugin**
After installing and activating this plugin, the last edit date, time, and timezone (based on the website's settings) will automatically print to posts and pages.

**To disable this plugin for a specific post type**
1. Go to Settings > Simply Show Last Edit Date
2. Check off the box next to "Disable for all <post type>"
3. Save changes
